  │  样例7-2: Controller 层文件结构                                         │

  │    PUT update(id,dto) / DELETE delete(id)                               │
  │  StudentController.java       -- /api/students                         │

  │    GET scoreTrend(studentNo)  /  GET errorAnalysis(studentNo)           │

  │                                                                        │

  │    StudentClassService.java        -- 接口: 5个方法                     │
  │    StudentService.java             -- 接口: 9个方法 (CRUD+Analysis)    │
